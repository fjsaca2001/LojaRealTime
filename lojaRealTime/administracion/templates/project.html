CrowdSensing móvil permite recolectar datos utilizando los teléfonos inteligentes mediante los sensores con los que esta cuenta y posterior enviar los datos a los proveedores de servicios para trabajar con estos datos en temas estadisticos, finalmente se los retorna al usuario final como datos importantes, cuyo fin es el de informar de manera estadística mediante graficas o ilustraciones varias situaciones. La movilidad es un tema que siempre ha estado presente. Con una correcta recolección de datos e interpretación de estos se podrá obtener varios beneficios. En la ciudad de Loja existen aplicaciones dedicadas a la movilidad, junto a la empresa Kradac, los mismos que mantienen una de las principales aplicaciones (Ktaxi) se recolectará datos en tiempo real que con una arquitectura CrowdSensing Mobile (MCS) se podrá hacer esto posible.

Proceso del CrowdSensing


El proceso del CrowdSensing móvil parte desde un smartphone el mismo que cuenta con sensores de los cuales para temas de movilidad se enfoca en el GPS estos datos se recolectan y envian a una base de datos, pero el proceso no se detiene ahí, ya que se reutilizan estos datos para procesarlos y brindar graficas estadisticas con datos importantes para que finalmente el usuario aproveche al maximo estos datos y continuar con un ciclo de vida.
